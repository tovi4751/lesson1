# YamMed – אתר תדמיתי לחברת תרופות

## מטרת הפרויקט
פיתוח אתר תדמיתי לחברת תרופות בשם YamMed, המתמחה בייצור תרופות בטעמים מותאמים אישית לילדים.  
האתר נועד להציג מידע על החברה, קטלוג תרופות, טופס להזמנת תרופה מותאמת ודף שאלות נפוצות.

---

## טכנולוגיות
- HTML5  
- CSS3  

הפרויקט פותח כאתר סטטי, ללא שימוש ב־JavaScript.

---

## מבנה האתר
האתר כולל את הדפים הבאים:
- דף הבית  
- קטלוג תרופות  
- הזמנת תרופה מותאמת  
- שאלות נפוצות (FAQ)

---

## עקרונות פיתוח
- שימוש במבנה HTML תקני וסמנטי  
- עיצוב באמצעות CSS בלבד  
- פריסת כרטיסים בעזרת CSS Grid  
- התאמה למובייל (Responsive Design)  
- אחידות עיצובית בין כל דפי האתר  

---

## הוראות הפעלה
יש לפתוח את הקובץ `home.html` בדפדפן.  
אין צורך בהתקנות נוספות.
